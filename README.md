# 心仪读报

## 项目简介
心仪读报是一款简洁高效的在线报纸阅读工具，支持浏览多种报纸的电子版内容，提供舒适的阅读体验。

## 功能特点

### 核心功能
- **多报纸支持**：内置多家主流报纸，可随时添加新报纸
- **智能解析**：自动解析报纸网站，获取PDF文件和版面目录
- **网页兼容**：解析失败时自动切换到直接访问网页版
- **边侧栏导航**：鼠标悬停左侧自动显示，支持固定和自动隐藏
- **搜索功能**：快速搜索报纸列表中的报纸

### 界面控制
- **灵活布局**：可调整边侧栏宽度和触发距离
- **多种显示模式**：支持文本式列表和图片平铺两种显示模式
- **响应式设计**：适配不同屏幕尺寸

### 高级功能
- **添加新报纸**：支持自定义添加新的报纸源
- **强制网页模式**：可选择直接访问网页版报纸
- **设置管理**：丰富的设置选项，满足个性化需求

## 安装使用

### 安装方法
1. 下载安装包：[心仪读报安装包](http://www.eersoft.com/myapp/NewsPaperReader/publish.htm)
2. 双击安装包，按照提示完成安装
3. 安装完成后，桌面会生成应用图标

### 基本使用
1. **启动应用**：双击桌面图标启动心仪读报
2. **浏览报纸**：鼠标移动到屏幕左侧，边侧栏会自动显示
3. **选择报纸**：在报纸列表中点击选择感兴趣的报纸
4. **查看版面**：选择报纸后，下方会显示该报纸的版面列表
5. **阅读内容**：点击版面包名查看具体内容
6. **搜索报纸**：在搜索框中输入报纸名称进行快速查找

## 注意事项

### 存储空间
- 软件会在桌面上创建"心仪读报"文件夹，用于存储浏览过的报纸文件
- 建议定期清理不需要的文件，以节省存储空间
- 该文件夹中所有内容皆可删除，不会影响软件正常运行

### 缓存机制
- 由于缓存机制，可能出现今天看到的报纸还是昨天的情况
- 解决方法：删除桌面"心仪读报"文件夹中当前日期对应报纸的文件夹

### 网络连接
- 使用心仪读报需要稳定的网络连接
- 首次访问报纸时需要下载内容，可能需要一些时间

### 版权声明
- 心仪读报仅提供报纸内容的浏览功能
- 报纸内容的版权归原出版机构所有

## 技术原理

1. **访问报纸网站**：通过HTTP请求访问报纸的官方网站
2. **解析HTML**：智能分析网页结构，提取版面信息和PDF链接
3. **下载内容**：获取PDF文件或直接访问网页内容
4. **展示内容**：使用WebView2控件显示报纸内容
5. **缓存管理**：将内容缓存到本地，提高后续访问速度

## 常见问题

### Q: 为什么看不到最新的报纸内容？
A: 可能是缓存导致的，删除桌面"心仪读报"文件夹中对应日期的报纸文件夹，然后重新打开应用即可。

### Q: 如何调整边侧栏的显示灵敏度？
A: 在设置窗口中，调整"侧边栏触发距离"滑块，数值越大，需要鼠标更靠近屏幕边缘才会显示边侧栏。

### Q: 报纸内容显示异常怎么办？
A: 尝试点击"刷新"按钮重新加载版面信息，或检查网络连接是否正常。如果问题持续，可以在编辑报纸时启用"强制直接访问网页版"选项。

### Q: 如何添加新的报纸？
A: 点击边侧栏顶部的"添加"按钮，在弹出的对话框中输入报纸名称和网址，点击确定即可。如果需要，可以同时启用"强制直接访问网页版"选项。
